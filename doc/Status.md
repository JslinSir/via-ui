### via-status  状态显示
   一个缺省提示组件


### 示例代码
* 基本使用
```
   <via-status  errorType="noNetwork"/>
```
 
 


### 属性
| 属性 | 说明 | 类型 | 默认值 | 备注 | 
| --- | --- | --- | --- | --- |
| src | 加载远程图片 | String | '' | - |
| errorType | 错误类型 | String | noNetwork | 404 500 imgError networkError noCotent noNetwork noOrder noRecord |
| full | 是否全屏缺省页 | String | scaleToFill | - |
| errorText | 错误文案 | String | '' |- |
| zIndex | 层级 | Number | -1 | - |
