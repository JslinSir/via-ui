### via-radio  单选
  单选框


### 示例代码
* 基本使用
```
<via-radio   />
```
*  自定义 大小颜色
```
	<via-radio  actionColor="#FF2332" radioSize="60"  action bind:onClick="handleClick"/>
```
 
 


### 属性
| 属性 | 说明 | 类型 | 默认值 | 备注 |
| --- | --- | --- | --- | --- |
| radioSize | 单选尺寸 | Number | 40 rpx|  |
| action | 选中状态 | Boolean | false | |
| actionColor | 选中颜色 | String| #1976D2 | | |
