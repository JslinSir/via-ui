### 示例代码

		<via-icon name="home" color="#091B4B" />

### 属性

| 属性 | 说明 | 类型 | 默认值 | 备注 |
| --- | --- | --- | --- | --- |
| size | 图标尺寸 | Number | 28 rpx |   |
| color | 图标颜色 | String | #CCCCCC |   |
| src | 远程图标 | String | '' |   | |
