### 水印 Watermark  
   水印组件


### 示例代码
* 基本使用
  
```
wxml:
	<via-watermark markStr="Via UI">
			<image class="img" src="./test.png" />
		</via-watermark>
 

```
 
 

### 属性
| 属性 | 说明 | 类型 | 默认值 | 备注 |
| --- | --- | --- | --- | --- |
| markStr |  水印文字  |  String |   |  |
| fontSize |  键盘 水印字体大小 |  Number | 32 | rpx单位 |
| spaceWidth |  间隔宽度 rpx |  Number | 300 |  rpx单位  |
| spaceHeight |  间隔高度   |  Number | 180 | rpx单位  |
