### 消息提示 Message
  消息提示组件


### 示例代码
* 基本使用
  
```
wxml：
<via-message id="message"  />
js：
this.selectComponent('#message').toast('Via UI')
this.selectComponent('#message').success('感谢使用！')
this.selectComponent('#message').error('提交失败')
this.selectComponent('#message').loading('加载中')
```
